Product Overview

Two tools.
One AI.

⌨
Claude Code

Claude Code is an agentic CLI tool installed directly on your machine. It can read your entire codebase, edit files, run shell commands, execute tests, and use external tools — operating as a true autonomous development partner inside your workflow.

npm i -g @anthropic-ai/claude-code Agentic Codebase-Aware MCP Support
$ claude "fix the failing auth tests"
Reading 47 files in src/auth…
Identified root cause in middleware.ts
✓ 3 tests passing — committed to branch fix/auth
💬
Claude Chat

Claude Chat is the web interface at claude.ai — a polished, accessible conversation experience for anyone. Upload documents, generate code snippets, brainstorm ideas, draft writing, or ask any question. No installation, no configuration.

Key Capabilities

What each one
does best

01
Autonomous Bug Fixing

Tell Claude Code to "fix the failing tests" and it reads the relevant files, identifies the root cause, writes the fix, runs the tests, and commits — all without you guiding every step.

02
Whole-Codebase Refactoring

Rename a function across 200 files, migrate an API, or restructure a directory. Claude Code can traverse your entire project and apply consistent changes with surgical precision.

03
MCP & External Tools

Connect Claude Code to databases, APIs, custom scripts, and external services via Model Context Protocol servers — expanding what it can see and do beyond your local machine.

04
Document Analysis

Upload PDFs, spreadsheets, images, or research papers and ask Claude Chat to summarize, extract insights, compare sections, or answer specific questions about the content.

05
Writing & Creative Work

Draft emails, write essays, generate marketing copy, ideate story plots, translate text, or polish prose. Claude Chat's conversational interface makes iterative creative work feel natural.

06
Instant Accessibility

No installation, no API key, no configuration. Open a browser, create an account, and you're talking to Claude in under a minute — the lowest-friction AI assistant available.

Decision Guide

Which one
should you use?

Use Claude Code when…
  • You're working inside an existing codebase and need the AI to understand the full project context
  • You want the AI to autonomously execute multi-step development tasks without hand-holding
  • You need to run tests, execute scripts, or issue shell commands as part of the workflow
  • You're doing large-scale refactoring or migrations across dozens or hundreds of files
  • You want to connect external tools, APIs, or databases via MCP servers
  • You prefer staying in your terminal or IDE rather than switching to a browser
  • You're building CI/CD pipelines or automating developer workflows at scale
Use Claude Chat when…
  • You need a quick answer, explanation, or analysis without any setup overhead
  • You're working on writing, research, ideation, or non-code creative tasks
  • You want to upload a document or image and ask questions about its contents
  • You're learning something new and prefer a back-and-forth conversational format
  • You're sharing access with non-technical teammates who don't use a terminal
  • You want rendered outputs — formatted documents, charts, visual artifacts
  • You're on a mobile device or a machine where you can't install software
